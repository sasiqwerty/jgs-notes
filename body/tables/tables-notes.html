Tables

Tables are used to tabulate and organise related data.

Example Table

This is a caption to the table given below
th1 th2 th3 th4
td1 td2 td3 td4
td1 td2 td3 td4

Instructions to create a table

  • Count the number of rows in the table
  • Count the number of table headers in the table
  • Count the number of table data points in the row
  • Create the number of rows necessary using the copy paste method
  • Insert the table headers in the first row and fill the contents
  • Insert the table data points in the rest of the rows and fill the contents accordingly
