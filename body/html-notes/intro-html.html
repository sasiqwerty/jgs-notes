Markup Language

Mark up is plain text with special symbols, the special symbols tell the browser/user to display/understand the document and its structure.

Why HTML is important

  • HTML gives structure to the web document, by converting plain text into something more meaningful using tags.
  • CSS adds style to the web document, its has nothing to do with the structure of the document.
  • Attributes of the elements have many uses, they can act as hooks to apply CSS,JS and many more functions
  • Cross-referencing the text using hyperlinks makes the internet more connected. Any part of the web can be connected using the hyperlinks making HTML that much more effective.
  • Using JavaScript we make the web-document dynamic and interactive.

Elements,tags and Attributes

  • An element is a part of a webpage.
  • In XML and HTML, an element may contain a data item or a chunk of text or an image, or perhaps nothing.
  • A typical element includes an opening tag with some attributes, enclosed text content, and a closing tag.
  • Elements and tags are not the same things. Tags begin or end an element in source code, whereas elements are part of the DOM, the document model for displaying the page in the browser.

Guidelines and Rules

  • HTML is not case sensitive
  • always close tags
  • Enclose the values of attributes in quotes
  • Attributes values can be case sensitive
  • Nest the elements in the right way
  • tag names don't contain spaces
  • the angle bracket notation has no spaces

Basic tags

HTML tag

The html tag ensures the browser reads the code as html and not as plain text.

head

The head tag contains information about the web document but contains no real text, holds links to style documents and scripts in some cases.

title

Holds the title of the document, it is displayed on the title bar at top of the window.

body

The body tag contains the main content of the document. There can be only one body element in a document.

Anchor tag

The HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each should indicate the link's destination.

h1 to h6

The h1 to h6 HTML elements represent six levels of section headings. h1 is the highest section level and h6 is the lowest.

p

HTML paragraphs can be any structural grouping of related content, such as images or form fields. Paragraphs are block-level elements, and notably will automatically close if another block-level element is parsed before the closing.
