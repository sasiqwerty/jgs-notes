HTML and CSS

Hypertext

A system allowing extensive cross-referencing between related sections of text and associated graphics.

Markup Language

Mark up is plain text with special symbols, the special symbols tell the browser/user to display/understand the document and its structure.

Why HTML is important

  • HTML gives structure to the web document, by converting plain text into something more meaningful using tags.
  • CSS adds style to the web document, its has nothing to do with the structure of the document.
  • Attributes of the elements have many uses, they can act as hooks to apply CSS,JS and many more functions
  • Cross-referencing the text using hyperlinks makes the internet more connected. Any part of the web can be connected using the hyperlinks making HTML that much more effective.
  • Using JavaScript we make the web-document dynamic and interactive.
