Lists

Lists are used to group related items together.
Ordered lists are used to create numbered lists while lists that are not in specific order can be created with the unordered list tag.

Example code for unordered lists

<ul>
        <li></li>
      </ul>

Example code for ordered lists

<ol>
        <li></li>
      </ol>
