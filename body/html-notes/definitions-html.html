Definitions in HTML

Elements
The elements are the building blocks of webpages, nesting the elements is the reason why HTML is so versatile and easy to use.
Tags
Tags are the delimiters that define the boundaries of elements in the source code.
Attributes
Attributes change/tweak the properties of the content add extra functionality and to the elements.
hyperlinks
The links that connect one part of the text document to the other, makes it easier for the user to navigate and author to set the flow.
hypertext
Plain text with hyperlinks.
Website
A website consists pages of HTML documents, giving rise to a meaningful structure
Webpage
A single functional part of a webpage meant for singular function.
HTML document
A plain text document with special symbols designed to be understood by the browser.
DOM - Document Object Model
DOM is a interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."
